# TERRA AI Model Registry Service

This is the documentation site for the TERRA AI Model Registry Service. The service is part of the wider [TERRA](https://terra-horizon.eu/) platform. This is part of the overall [platform documentation](https://terra-horizon.github.io/).

The AI Model Registry service integrates inference models and exposes them via endpoints.

You can use the menu options on the left to navigate through the available documentation. You may be interested to jump directly to an [Architecture Overview](architecture.md), see the available [Onboarding Material](onboarding.md). You can find here the service [code repository](https://github.com/terra-horizon/ai-model-registry).

For any questions, you can consult the [FAQ](faq.md), check if there is a relevant [issue](https://github.com/terra-horizon/ai-model-registry/issues) answering your question or contact the [TERRA Help Desk](https://terra-horizon.eu/contact-us/).
